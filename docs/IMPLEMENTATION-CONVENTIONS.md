# Implementation Conventions

## Linguagem

Este projeto e `100% JavaScript`.

- nao usar TypeScript
- nao adicionar `tsconfig.json`
- nao adicionar tipagem como parte obrigatoria do scaffold

## Documentacao no codigo

O codigo deste projeto deve ser comentado acima do usual.

### Regras

- cada arquivo de codigo abre com um comentario curto explicando o papel do arquivo
- cada funcao relevante recebe comentario proprio
- comentarios devem explicar responsabilidade, entradas, saidas e contexto
- quando existir documentacao tecnica complementar, o codigo deve apontar para ela por comentario

## Objetivo

Permitir que outra IA ou dev humano entenda o sistema com o minimo de leitura implícita.
